# Covid-19
Python3 Project to Generate Graphs and stuff about Covid-19

## How to Use
Pretty Easy to use, just run `python3 Covid19.py` command in your Terminal or CMD.  
Try Running : `python3 Covid19.py -h` to view essential options.

## Prerequisites
Python3
